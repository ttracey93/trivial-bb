# trivial-bb

Get Started
-----------------------
1. Clone the repository
2. npm install
3. install bower ( npm install -g bower )
4. Installs the project dependencies ( bower install )
5. grunt server


Note
-----------------------
Google Map API will only work in the hostname in
the bower.json file is set to '0.0.0.0'.
